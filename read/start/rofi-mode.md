## 啟動模式

| 內建的啟動模式     | 啟動指令                   |
| ---------------- | ------------------------ |
| **run**          | `rofi -show run`         |
| **drun**         | `rofi -show drun`        |
| **window**       | `rofi -show window`      |
| **ssh**          | `rofi -show ssh`         |
| **keys**         | `rofi -show keys`        |
| **filebrowser**  | `rofi -show filebrowser` |
| **windowcd**     | `rofi -show windowcd`    |
| **combi**        | `rofi -show combi`       |
