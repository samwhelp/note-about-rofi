## 啟動模式

下面三個，是我個人比較常用「啟動模式」。

| Mode       | Command             |
| ---------- | ------------------- |
| **run**    | `rofi -show run`    |
| **drun**   | `rofi -show drun`   |
| **window** | `rofi -show window` |


接著可以參考[我個人慣用的「按鍵綁定」來「啟動 Rofi」](https://samwhelp.github.io/note-about-rofi/read/start/keybind-launching-rofi.html)。
